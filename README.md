# 🏦 Banking CRUD API

A robust RESTful API developed for a banking application.  
This project serves as the backend foundation for a full **CRUD** system that manages bank transactions and customer data.

---

## 📖 About the Project

The Banking CRUD API is designed to handle essential banking operations through a secure and efficient RESTful architecture.  
It acts as the backend layer for a modern web application, exposing endpoints for managing customers and transactions.

The backend is built with **ASP.NET Core** and uses **PostgreSQL** for reliable and performant data storage.

---

## 🎯 Project Goal

The goal of this project is to provide:
- A clean and usable backend architecture
- Secure handling of banking-related data
- A solid foundation for frontend integration
- Best practices for RESTful API development

---

## 🛠️ Technology Stack

### Backend
- **Framework:** ASP.NET Core 
- **Database:** PostgreSQL  
- **ORM:** Entity Framework Core  
- **Authentication:** JWT / ASP.NET Identity (configurable)

### Frontend
- **Framework:** React  
- **Related Project:** `FrontBankingApp`

---

## ⚙️ Features

- Full CRUD operations for banking entities
- RESTful API design
- Secure authentication and authorization
- PostgreSQL data persistence
- Scalable and maintainable project structure
